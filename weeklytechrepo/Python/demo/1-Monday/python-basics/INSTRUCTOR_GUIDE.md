# Demo: Python Basics - Variables and I/O

## Phase 1: The Concept

**Time:** 5 mins

1. **Context:** Welcome to Python! Explain that unlike Java, Python is an interpreted, dynamically-typed language. There is no `public static void main(String[] args)` required just to print a line of text.
2. **The Simplicity of Syntax:** Show how indentation replaces curly braces `{}` and how variables don't require explicit type declarations like `int x = 5`.

## Phase 2: The Code (Live Implementation)

**Time:** 15 mins

1. Open `code/basics.py`.
2. **Walkthrough the Code:**
    * Highlight the variable creation. The interpreter infers the datatype based on the assigned value.
    * Show `print()` and the modern f-string syntax `f"Hello {name}!"`. Explain how much cleaner this is than Java's string concatenation.
    * Demonstrate `input()`. Note that `input()` *always* captures data as a String, even if the user types a number.
3. **Run the App:**
    * Execute the script using `python basics.py` in the terminal.
    * Interact with the prompt to show the output.
