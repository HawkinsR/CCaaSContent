# Demo: Lists and Dictionaries

## Phase 1: The Concept

**Time:** 5 mins

1. **Context:** We know how to store a single string or number. What if we need to store 100 Usernames? What if we need to store a user's First Name, Last Name, and Age grouped together?
2. **The Collections:** Introduce the `list` (ordered, mutable array) and the `dict` (key-value pairs, comparable to a Java HashMap or JSON object).

## Phase 2: The Code (Live Implementation)

**Time:** 15 mins

1. Open `code/lists_dicts.py`.
2. **Walkthrough the Code:**
    * **Lists:** Show the `[]` syntax. Emphasize Zero-Indexing. Demonstrate `.append()` and `.remove()`. Show Python's powerful slicing syntax `[0:2]`.
    * **Dictionaries:** Show the `{}` syntax. Emphasize that we access data by `key` (e.g., `"name"`), not by numerical index. Show how to add a completely new key-value pair on the fly.
3. **Run the App:**
    * Execute the script.
    * *Instructor Note:* Tell them they will combine these structures shortly to build an in-memory database.
